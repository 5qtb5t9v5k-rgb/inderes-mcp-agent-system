# Streamlit Cloud uses uv pip install -r requirements.txt.
#
# Two pitfalls:
#   1. uv defaults to stable-only resolution. We need pre-releases because
#      agent-framework-gemini is still alpha (no stable release exists).
#      Configured via `[tool.uv] prerelease = "allow"` in pyproject.toml.
#   2. The PyPI distribution `agent-framework` is a META package containing
#      only `agent_framework_meta/__init__.py`. The actual `agent_framework`
#      Python package (with `BaseChatClient` etc.) ships in `agent-framework-core`.
#      Depending on `agent-framework` is enough to pull `agent-framework-core`
#      transitively, but be aware that the source code lives in core.

# Real Python package containing agent_framework/__init__.py
agent-framework-core
